==== Heading_2 ====
SHORT_EDGES (Short Edges)

==== Body ====
Lists the edges in the model with a length smaller than a specified limit. You can define the limit for the length using the SHORT_EDGE constant configuration option. If any short edges are found, the features in question can be highlighted in the Creo Parametric window.
You can also access the Global Reference Viewer from the Creo ModelCHECK report to view all references to and from the reported features.
The value for SHORT_EDGE can be set to CALCULATE. If this is the case, the value used to determine a short edge is based on the size of the model.
